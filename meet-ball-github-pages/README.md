# Meet Ball + LANE LAB

볼링 정기전 운영 페이지와 AI 볼링 분석 도구를 한 사이트로 합친 정적 웹앱입니다.

## 주요 기능

- 공지, 점수 입력, 순위, 회차 기록
- 선수·회비·투표·팀 편성·레인 배정
- AI 자세 분석
- 공 궤적과 볼 스피드 분석
- 오일 패턴 이미지 분석 및 레인 오버레이
- 투구 후 브레이크 포인트·진입 보드·진입각 표시

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더 안의 모든 파일을 저장소 최상단에 업로드합니다.
3. 저장소 `Settings > Pages`로 이동합니다.
4. `Deploy from a branch`를 선택합니다.
5. 브랜치는 `main`, 폴더는 `/(root)`를 선택하고 저장합니다.

배포 주소는 보통 다음 형식입니다.

`https://사용자이름.github.io/저장소이름/`

카메라 기능은 GitHub Pages의 HTTPS 주소에서 사용할 수 있습니다.

## 폴더 구조

```text
index.html       정기전 메인 페이지
ai/              LANE LAB AI 분석 웹앱
.nojekyll        GitHub Pages 정적 파일 설정
```

## 참고

- 사용자 기록은 기본적으로 브라우저의 로컬 저장소에 보관됩니다.
- 다른 휴대폰이나 브라우저에서는 로컬 기록이 자동 공유되지 않습니다.
- AI 모델과 OCR 모델은 최초 사용 시 인터넷에서 내려받습니다.

## 공개 배포 전 보안 확인

- 이 GitHub 공유본에서는 하드코딩된 관리자 비밀번호를 제거했습니다.
- 관리자 기능이 필요하면 Firebase Authentication을 연결해야 합니다.
- Firebase API 키 자체는 공개될 수 있지만, Realtime Database 보안 규칙이 반드시 읽기·쓰기를 제한해야 합니다.
- 현재 HTML에는 회원 이름, 점수, 회비 등 클럽 데이터가 포함될 수 있습니다. 공개 저장소에 올리기 전에 공개 범위를 확인하세요.
